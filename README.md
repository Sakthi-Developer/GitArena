# GitArena

Interactive Git learning through visual challenges. Master branching, merging, rebasing, and more.

## Setup

```bash
cd gitarena
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Test

```bash
npm test
```

## Benchmark

```bash
npm run bench
```

## Routes

- `/` — Home page with level selection
- `/play/:levelId` — Play a specific level
- `/editor` — Free sandbox mode
